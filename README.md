# Atishoo

Augment github issues with some extra stuff.

## Setup

Requires npm and nvm, mongodb

```
nvm install iojs-v2.4.0
npm install -g nodemon
npm i
```


## Develop

```
npm run dev
```

## Test

```
npm test
```
